# SYNONYMS Flashcard Game

An interactive flashcard study tool based on the Cabrera Research Lab's SYNONYMS framework.

## How to Play

1. A word appears on the card
2. Select which category you think it belongs to: **Distinctions**, **Systems**, **Relationships**, or **Perspectives**
3. Click the card to reveal the answer
4. Track your accuracy as you go

## Play Now

👉 [https://phil-rev.github.io/synonyms-flashcards](https://phil-rev.github.io/synonyms-flashcards)

## About

The game contains ~300 words across four DSRP categories. It's a single self-contained HTML file with no dependencies — just open `index.html` in any browser. Supports light and dark mode.
